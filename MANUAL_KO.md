# 프로그램램 사용 매뉴얼

이 매뉴얼은 처음 사용하는 분도 순서대로 따라 할 수 있도록 작성한 안내서입니다.  
작업 순서는 **로그인 → 유닛 관리(필요 시 길이 프리셋 등록, 유닛 생성) → 프로젝트 생성 → 도면 업로드 → 도면 위에 철근 입력 → 계산 결과 확인 → 인쇄** 순서입니다.

## 1. 로그인하기

1. 시스템 주소에 접속합니다.
2. 로그인 화면에서 아래 계정 정보를 입력합니다.
   - 이메일: `admin907@gmail.com`
   - 비밀번호: `password`
3. `ログイン` 버튼을 누릅니다.
4. 로그인에 성공하면 프로젝트 목록 화면이 열립니다.

로그인이 되지 않으면 이메일과 비밀번호를 다시 확인한 뒤, 담당자에게 문의해 주세요.

## 2. 유닛 관리 들어가기

도면에 철근을 그리기 전에 먼저 유닛을 만들어 둡니다.  
유닛은 `外周`, `内`처럼 도면에서 사용할 철근 형상과 계산 기준을 저장해 두는 항목입니다.

1. 화면 상단의 `ユニット管理`을 클릭합니다.
2. 유닛 목록 화면이 열립니다.
3. 새 유닛을 만들려면 `+ 新規作成` 버튼을 누릅니다.

## 3. 유닛 만들기

### 3.1 기본 정보 입력

1. **유닛명**을 입력합니다.  
   - 예: `外周`, `内`
2. **표시색**을 선택합니다.  
   - 도면 위의 선 색·결과 화면에서 유닛을 구분하는 데 쓰입니다.  
   - 다른 유닛에서 이미 쓰는 색은 선택할 수 없습니다.
3. **形状プリセット**에 저장된 항목이 있으면, **読み込み**를 눌러 **미리 만들어 둔 형상**을 이 유닛에 바로 불러다 쓸 수 있습니다(목록이 비어 있으면, 먼저 프리셋을 추가해 두어야 합니다).

### 3.2 형상 그리기

1. 형상 입력 영역에서 **시작점**을 클릭합니다.
2. **다음 점**을 클릭해 이어질 선을 만듭니다. 꺾이는 곳이 있으면 점을 찍는 것을 반복해 윤곽을 완성합니다.
3. 자료와 **같은 모양**인지 확인합니다.
4. **Shift**를 누른 채로 그리면, 선이 **수평·수직**에만 맞게 그려집니다. ㄱ자·직각 형상에 편리합니다.

### 3.3 철근 배치 입력

1. 유닛 상세 화면에서 **철근 위치**를 추가합니다.
2. **D10**, **D13** 등 철근 공칭(종류)을 지정합니다.
3. 필요한 **수량**만큼 철근을 배치합니다.

### 3.4 구간 길이(수치)및 간격선 입력

같은 캔버스에서 **입력 방식**이 둘로 나뉩니다.

- **한 번만 클릭**한 뒤에 그 위치(구간)에 **길이 수치**를 입력할 수 있습니다.
- **누른 채로 드래그**하면 **간격선**이 그어집니다(피치·간격이 적용될 가이드).

간격선을 그린 뒤, **각 직선 구간에 대응하는 길이**를 넣습니다. 이 수치가가 **형상 길이(결과 카드의 왼쪽 쪽)**를 계산할 때 쓰입니다.

- 형상을 보며 **구간마다** 자료에 적힌 길이를 입력합니다.
- **형상에 실제로 쓰는** 치수만 넣고, 참고용·보조 치수는 우측의 形状長さから除外する체크를 눌러 제외합니다.

**입력·계산 시 참고**

- **포함 예**: `515`, `130`, `220`, `400` 처럼, 실제 길이에 더해지는 수치
- **제외 예**: 형상 합산에 쓰지 않는 **참고·보조** 치수
- L자 **꺾임**이 있으면 **`L字本数`**를 입력합니다. 값이 1 이상이면, 전체 길이에서 **20mm가 1회** 차감된 것으로 반영됩니다.
- 수치가 어긋나면 **결과 카드의 왼쪽 길이**가 달라질 수 있으니, **형상 그리기 → 구간 길이 입력** 순서로 맞춰 두는 것이 좋습니다.

**계산 요약 예**

```text
外周: 515 + 130 + 220 + 400 - 20 = 1245
内:   345 + 200 - 20 = 525
```


### 3.6 피치 입력

1. 피치란에 **`@200`**처럼 입력합니다. 이 값이 フック 例: 1,245 × 202의 **오른쪽 개수**를 계산하는 데 사용됩니다.
2. **피치**가 비어 있으면, 유닛이이 저장이 되지 않습니다.

### 3.7 저장

1. 입력이 끝났으면 **`保存`**를 누릅니다.  
2. **유닛 목록**에 이 유닛이 보이면 등록이 완료된 것입니다.

### 3.8 길이 프리셋(長さプリセット) 등록

도면에서 선을 그을 때 **자주 쓰는 길이**를 골라 넣을 수 있도록, 길이 묶음을 **미리 등록**해 두는 항목입니다(프로젝트·도면 편집에서 씁니다).
**프로젝트를 만들기 전**에, 유닛 관리 화면에서 등록해 두는 것이 좋습니다.

1. **`ユニット管理`**을 연 상태에서, 오른쪽 위 **`+ 新規作成`** 옆에 있는 **`長さプリセット`**을 누릅니다.
2. **등록한 길이 그룹**이 나열됩니다. 아직이면 **`追加`** → **`長さプリセットを追加`** 화면이 열립니다.
3. **プリセット名**(필수)과, 필요하면 **説明(任意)**을 넣고, **長さ一覧**에서 **番号**·**長さ（mm）** 를 행마다 입력합니다. 행이 부족하면 **`+ 行を追加`**로 늘릴 수 있습니다(한 줄이라도 **長さ（mm）** 는 1개 이상 필요합니다).
4. **`保存`**로 등록을 마칩니다(목록에서 **更新**·**削除**로 이후에 바꿀 수 있습니다).
5. 나중에 **도면 편집**에서 유닛을 쓰면, 등록한 프리셋의 길이를 **길이 후보**로 선택하며 그릴 수 있습니다.

## 4. 프로젝트 만들기


유닛을 만든 뒤, 실제 도면과 계산 결과를 관리할 프로젝트를 만듭니다.

1. 프로젝트 목록 화면으로 이동합니다.
2. `+ 新規プロジェクト` 버튼을 누릅니다.
3. 프로젝트명을 입력합니다.
4. 필요하면 설명을 입력합니다.
5. 저장하면 프로젝트가 생성됩니다.
6. 생성된 프로젝트를 클릭해 상세 화면으로 들어갑니다.

## 5. 도면 업로드하기

1. 프로젝트 상세 화면에서 도면 업로드 영역을 찾습니다.
2. 도면 파일을 업로드합니다.
   - 사용 가능 형식: PDF, PNG, JPG
3. 업로드가 끝나면 자동으로 도면 편집 화면으로 이동합니다.

## 6. 도면 위에 철근 입력하기

도면을 업로드한 뒤, 미리 만들어 둔 유닛과 長さプリセット를 선택해 도면 위에 철근 선분을 입력합니다.

1. 도면 편집 화면을 엽니다.
2. アクティブユニット에서 사용할 유닛 또는 색상을 선택합니다.
3. 유닛을 선택하면 나오는 長さプリセット의 창에서 원하는 프리셋이 있으면 고릅니다. 
3. 도면 위에서 철근 구간을 클릭하거나 드래그해 선을 그립니다. (Shift 키를 누른 상태로 선을 그리면 선이 수평 또는 수직으로만 그려집니다.)
4. 長さを選択 창이 뜨면 선택하고싶은 길이의를 선택합니다 (키보드 번호키를 눌러 빠르게 입력력할수있습니다.) 
5. 필요한 모든 구간을 같은 방식으로 입력합니다.

도면 위에 그린 선분은 계산 결과에 그대로 반영됩니다.  
따라서 길이, 색상, 유닛 선택이 맞는지 입력 후 꼭 확인해 주세요.
확인이 끝나면 우측에있는 절단 계산 버튼을 누르세요.

## 7. 입력 데이터 확인하기

계산 화면의 `入力データ` 영역에서 입력된 내용을 확인합니다.

확인할 내용:

- `外周`, `内` 등 구분이 맞는지
- 길이가 맞는지
- 각 길이별 본수가 맞는지
- 색상이 맞는지
- 전체 부재 수가 맞는지

입력 데이터가 다르면 도면 화면으로 돌아가 선분을 수정한 뒤 다시 계산합니다.

## 8. 고객 정보 입력하기

계산 화면의 `顧客情報` 영역에서 출력물에 표시할 정보를 입력합니다.

입력 항목:

- 회사명
- 주소
- 고객명

인쇄할 때 고객 정보는 입력 데이터보다 위쪽에 표시됩니다.

## 9. 계산 실행하기

1. `計算設定` 영역을 확인합니다.
2. `元材長さ(mm)`에 원자재 길이를 입력합니다.
   - 일반적으로 `6000`을 사용합니다.
3. `部材長さ補正(mm)`을 입력합니다.
   - 예: `-30`
   - 마이너스 값은 길이를 짧게 계산합니다.
   - 플러스 값은 길이를 길게 계산합니다.
4. `計算を実行` 버튼을 누릅니다.
5. 계산이 완료되면 결과가 화면 아래에 표시됩니다.

## 10. 계산 결과 확인하기

### 10.1 철근 종류별 필요 본수

`鉄筋種類別の必要本数` 표에서 길이별 필요 본수를 확인합니다.

표에서 확인할 내용:

- `長さ(mm)`: 절단에 사용할 길이
- `D13`, `D10`: 철근 종류별 필요 수량
- `合計`: 해당 길이의 전체 수량

### 10.2 フック付 결과 카드

`フック付` 영역에서는 유닛별 결과를 확인합니다.

예:

```text
フック付
外周
1,245 × 202

フック付
内
525 × 235
```

왼쪽 값:

- 유닛 형상 길이입니다.
- 예: `1,245`, `525`

오른쪽 값:

- 자료 기준 개수입니다.
- 예: `202`, `235`

개수는 길이를 100mm 단위로 반올림한 뒤 피치 기준으로 계산합니다.

예:

```text
4095는 4100으로 보고 계산
3640은 3600으로 보고 계산
2730은 2700으로 보고 계산
910은 900으로 보고 계산
690은 700으로 보고 계산
```

결과가 예상과 다르면 입력 데이터의 길이와 수량을 먼저 확인해 주세요.

## 11. 인쇄하기

1. 계산 결과 화면에서 `印刷` 버튼을 누릅니다.
2. 인쇄 미리보기 화면이 열립니다.
3. 출력 내용을 확인합니다.
4. 프린터로 출력하거나 PDF로 저장합니다.

### 날짜와 URL이 같이 찍히는 경우

인쇄 미리보기에서 날짜, 페이지 번호, 화면 주소가 함께 표시될 수 있습니다.  
이 경우 인쇄 설정에서 `머리글과 바닥글` 옵션을 끄면 사라집니다.

## 13. 전체 작업 순서 요약

1. 로그인합니다.
2. 유닛 관리에서 유닛을 만듭니다.
3. 유닛의 색상, 형상, 형상 프리셋, 피치, 철근 배치,길이 프리셋을 저장합니다.
4. 프로젝트를 만듭니다.
5. 도면을 업로드합니다.
6. 도면 위에 철근 선분을 그립니다.
7. 입력 데이터를 확인합니다.
8. 고객 정보를 입력합니다.
9. 계산 설정을 확인합니다.
10. `計算を実行` 버튼을 누릅니다.
11. 결과 표와 `フック付`값을을 확인합니다.
12. 필요하면 `印刷` 버튼으로 출력합니다.

## 14. 자주 확인해야 하는 부분

### 결과 카드의 개수가 예상과 다를 때

아래 내용을 확인합니다.

- 도면에 입력한 길이가 맞는지
- 길이별 수량이 맞는지
- 해당 선분의 유닛 색상이 맞는지
- 유닛의 피치가 맞는지

### 형상 길이가 예상과 다를 때

아래 내용을 확인합니다.

- 유닛 상세 화면의 치수가 맞는지
- 참고 치수가 계산에서 제외되어 있는지
- L자 꺾임이 있는 경우 `L字本数`가 입력되어 있는지

### 도면에서 원하는 유닛을 선택할 수 없을 때

아래 내용을 확인합니다.

- 유닛이 저장되어 있는지
- 유닛이 비활성화되어 있지 않은지
- 같은 표시색을 다른 유닛에서 이미 사용 중인지

### 인쇄물이 원하는 형태로 나오지 않을 때

아래 내용을 확인합니다.

- `印刷` 버튼을 눌러 인쇄했는지
- 인쇄 미리보기에서 배율이 적절한지
- 날짜나 URL이 보이면 인쇄 설정에서 `머리글과 바닥글`을 껐는지
