# 프로그램 사용 매뉴얼

이 문서는 로그인부터 유닛 등록, 도면 위 철근 입력, 계산 결과 확인, 인쇄 출력까지의 기본 사용 흐름을 설명합니다.

## 1. 로그인

1. 브라우저에서 시스템 주소를 엽니다.
2. 로그인 화면에서 아래 계정 정보를 입력합니다.
   - 이메일: `admin907@gmail.com`
   - 비밀번호: `password`
3. `ログイン` 버튼을 누릅니다.
4. 로그인에 성공하면 프로젝트 목록 화면으로 이동합니다.

로그인이 되지 않는 경우에는 이메일과 비밀번호를 다시 확인한 뒤, 관리자에게 문의합니다.

## 2. 프로젝트 만들기

1. 프로젝트 목록 화면에서 `+ 新規プロジェクト` 버튼을 누릅니다.
2. 프로젝트명을 입력합니다.
3. 필요하면 설명을 입력합니다.
4. 저장하면 프로젝트가 생성됩니다.
5. 생성된 프로젝트를 클릭해 프로젝트 상세 화면으로 들어갑니다.

## 3. 도면 업로드

1. 프로젝트 상세 화면에서 도면 업로드 영역을 찾습니다.
2. PDF, PNG, JPG 형식의 도면 파일을 업로드합니다.
3. 업로드가 완료되면 도면 목록에 파일이 표시됩니다.
4. 도면을 클릭하면 도면 편집 화면으로 이동합니다.

주의:
- 한 프로젝트에 이미 도면이 등록되어 있는 경우 추가 업로드가 제한될 수 있습니다.
- 도면이 잘 보이지 않으면 원본 파일 해상도나 PDF 페이지를 확인합니다.

## 4. 유닛 관리 들어가기

1. 상단 메뉴에서 `ユニット管理`을 클릭합니다.
2. 유닛 관리 화면에서 기존 유닛 목록을 확인합니다.
3. 새 유닛을 만들려면 `+ 新規作成` 버튼을 누릅니다.

유닛은 도면 위에 그릴 철근의 종류, 표시색, 형상 길이, 피치, 배근 정보를 저장하는 기준 데이터입니다.

## 5. 유닛 생성

### 5.1 기본 정보 입력

1. `ユニット新規作成` 화면에서 유닛명을 입력합니다.
2. 표시색을 선택합니다.
   - 다른 유닛에서 이미 사용 중인 색은 선택할 수 없습니다.
   - 표시색은 도면 위 선 색상과 결과 집계 구분에 사용됩니다.
3. 필요한 경우 번호, 위치, 형상 타입 등을 설정합니다.

### 5.2 형상과 치수 입력

1. 상세/형상 편집 영역에서 유닛 형상을 그리거나 템플릿을 선택합니다.
2. 형상 길이에 포함될 치수를 입력합니다.
3. 참고 치수는 결과 길이에 포함되지 않도록 제외 처리합니다.
4. L자 꺾임이 있는 경우 `L字本数`를 입력합니다.
   - `L字本数`가 1 이상이면 형상 길이 합계에서 20mm가 한 번 차감됩니다.

예:
- 外周: `515 + 130 + 220 + 400 - 20 = 1245`
- 内: `345 + 200 - 20 = 525`

### 5.3 피치 입력

1. 피치 입력 영역에서 `@200`처럼 피치를 입력합니다.
2. 피치는 결과 카드 오른쪽 개수 계산에 사용됩니다.
3. 피치가 비어 있으면 계산 대상에서 제외될 수 있습니다.

### 5.4 철근 배치 입력

1. 유닛 상세 캔버스에서 철근 위치를 추가합니다.
2. D10, D13 등 철근 종류와 수량이 유닛에 저장됩니다.
3. 저장 버튼을 눌러 유닛을 저장합니다.

저장 후 유닛 목록에 새 유닛이 표시되면 등록이 완료된 것입니다.

## 6. 도면 위에 철근 그리기

1. 프로젝트 상세 화면에서 도면을 엽니다.
2. 도면 편집 화면에서 사용할 유닛 또는 색상을 선택합니다.
3. 도면 위에서 철근 구간을 클릭하거나 드래그해 선분을 추가합니다.
4. 선분 길이를 입력하거나, 저장된 길이 프리셋/유닛 길이를 선택합니다.
5. 필요한 선분을 모두 입력합니다.

도면 위 선분은 다음 계산에 사용됩니다.
- 철근 종류별 필요 본수
- フック付 결과 카드
- 절단 최적화 결과

## 7. 도면 입력 데이터 확인

도면 또는 계산 화면의 입력 데이터 영역에서 다음 내용을 확인합니다.

- 外周, 内 등 유닛별 그룹
- 선분 길이
- 각 길이별 본수
- 색상 구분
- 전체 부재 수

입력 데이터가 기대와 다르면 도면 편집 화면으로 돌아가 선분 길이, 유닛, 색상, 수량을 수정합니다.

## 8. 절단 계산 실행

1. 프로젝트의 계산 화면으로 이동합니다.
2. `計算設定`에서 원자재 길이를 입력합니다.
   - 기본값 예: `6000`
3. 부재 길이 보정을 입력합니다.
   - 예: `-30`
   - 마이너스 값은 입력 길이를 짧게, 플러스 값은 길게 계산합니다.
4. `計算を実行` 버튼을 누릅니다.

계산이 완료되면 결과 화면이 표시됩니다.

## 9. 계산 결과 확인

### 9.1 철근 종류별 필요 본수

`鉄筋種類別の必要本数` 표에서는 길이별로 D13, D10 등의 필요 본수를 확인할 수 있습니다.

표 구성:
- `長さ(mm)`: 보정 적용 후 절단 대상 길이
- `D13`, `D10`: 철근 종류별 필요 수량
- `合計`: 해당 길이의 총 필요 수량

### 9.2 フック付 결과 카드

`フック付` 영역에는 유닛별 형상 길이와 개수가 표시됩니다.

예:

```text
フック付
外周
1,245 × 202

フック付
内
525 × 235
```

왼쪽 값:
- 유닛 형상 길이입니다.
- 저장된 형상 길이 또는 annotation 치수 합계를 기준으로 합니다.
- L자 꺾임이 있으면 20mm가 한 번 차감됩니다.

오른쪽 값:
- 자료 기준 개수입니다.
- 길이를 먼저 100mm 단위로 반올림한 뒤 피치로 나눠 계산합니다.

계산 방식:

```text
normalizedLength = Math.round(lengthMm / 100) * 100
baseCount = Math.round(normalizedLength / pitchMm)
```

예:

```text
4095 -> 4100 -> 4100 / 200 = 20.5 -> 21
3640 -> 3600 -> 3600 / 200 = 18 -> 18
2730 -> 2700 -> 2700 / 200 = 13.5 -> 14
910 -> 900 -> 900 / 200 = 4.5 -> 5
690 -> 700 -> 700 / 200 = 3.5 -> 4
```

## 10. 고객 정보 입력

계산 화면의 `顧客情報` 영역에서 다음 정보를 입력할 수 있습니다.

- 회사명
- 주소
- 고객명

이 정보는 출력물에 함께 표시됩니다.

프린트 시에는 고객 정보가 입력 데이터보다 위에 표시되도록 설정되어 있습니다.

## 11. CSV 출력

1. 계산 결과 화면에서 `CSV出力` 버튼을 누릅니다.
2. 최적화 결과가 CSV 파일로 다운로드됩니다.
3. 필요하면 엑셀 등에서 열어 확인합니다.

CSV에는 전체 원자재 수, 폐기 길이, 폐기율, 철근 종류별 결과 등이 포함됩니다.

## 12. 인쇄 출력

1. 계산 결과 화면에서 `印刷` 버튼을 누릅니다.
2. 브라우저 인쇄 미리보기 화면이 열립니다.
3. 출력 내용을 확인합니다.
4. 프린터 또는 PDF 저장을 선택합니다.
5. 인쇄 또는 저장을 실행합니다.

프린트 전용 화면에서는 다음 항목을 숨기도록 설정되어 있습니다.

- 상단 시스템 메뉴
- 로그인 이메일
- `ユニット管理`
- `ログアウト`
- `プロジェクトに戻る`
- 프로젝트 제목과 `切断最適化の計算`
- `CSV出力`, `印刷` 버튼

프린트에는 회사 로고가 표시됩니다.

주의:
- 브라우저 인쇄 미리보기의 날짜, URL, 페이지 번호는 브라우저의 `Headers and footers` 옵션에서 표시됩니다.
- Chrome에서는 인쇄 창의 `More settings`에서 `Headers and footers` 체크를 끄면 날짜와 URL이 사라집니다.

## 13. 일반적인 작업 순서 요약

1. 로그인합니다.
2. 프로젝트를 생성합니다.
3. 도면을 업로드합니다.
4. `ユニット管理`에서 유닛을 만듭니다.
5. 유닛의 색상, 형상, 피치, 철근 배치를 저장합니다.
6. 도면 화면에서 유닛을 선택하고 철근 선분을 그립니다.
7. 선분 길이와 수량을 확인합니다.
8. 계산 화면에서 고객 정보와 계산 설정을 입력합니다.
9. `計算を実行`을 누릅니다.
10. 철근 종류별 필요 본수와 `フック付` 결과를 확인합니다.
11. 필요하면 CSV를 출력합니다.
12. `印刷` 버튼으로 출력합니다.

## 14. 문제가 생겼을 때 확인할 것

### 결과 카드 개수가 너무 크게 나오는 경우

- 유닛 철근 총수량이 개수에 중복으로 곱해지고 있지 않은지 확인합니다.
- 결과 카드 오른쪽 값은 `computedCount`가 아니라 `unitSummaryCount` 기준이어야 합니다.

### 형상 길이가 자료와 다른 경우

- 유닛의 annotation 치수가 정확한지 확인합니다.
- 참고 치수가 제외 처리되어 있는지 확인합니다.
- `L字本数`가 필요한 경우 입력되어 있는지 확인합니다.

### 도면에서 유닛이 선택되지 않는 경우

- 유닛이 저장되어 있는지 확인합니다.
- 유닛이 비활성화되어 있지 않은지 확인합니다.
- 같은 색상이 다른 유닛에서 이미 사용 중인지 확인합니다.

### 프린트에 날짜나 URL이 나오는 경우

- Chrome 인쇄 설정에서 `Headers and footers`를 끕니다.
- 앱 내부 요소가 아니라 브라우저가 자동으로 붙이는 정보입니다.
