# 操作マニュアル

本マニュアルは、初めて使う方でも手順を追って作業できるようにまとめた案内です。  
作業の流れは **ログイン → ユニット管理（必要に応じて長さプリセット登録）→ ユニット作成 → プロジェクト作成 → 図面アップロード → 図面上に鉄筋線分を入力 → 切断計算・計算結果の確認 → 印刷** です。

## 1. ログイン

1. システムのURLにアクセスします。
2. ログイン画面で次の情報を入力します。  
   - メール: `admin907@gmail.com`  
   - パスワード: `password`
3. **`ログイン`** を押します。
4. 成功すると **プロジェクト** 一覧（ダッシュボード）が開きます。

ログインできない場合は、メールとパスワードを再確認のうえ、担当者に問い合わせてください。

## 2. ユニット管理を開く

図面に鉄筋を描く前に、あらかじめ **ユニット** を登録しておきます。  
ユニットは `外周` や `内` のように、図面で使う断面の形・色・ピッチ・鉄筋の前提をまとめたデータです。

1. 画面上部の **`ユニット管理`** を開きます。
2. ユニットの一覧が表示されます。
3. 新規に作る場合は **`+ 新規作成`** を押します。

## 3. ユニットの作成

### 3.1 基本情報

1. **ユニット名**を入力します。  
   - 例: `外周`, `内`
2. **色**（表示色）を選びます。  
   - 図面の線の色や、計算画面での区切りに使います。  
   - 他のユニットで既に使われている色は選べません。
3. **形状プリセット** に保存済みのものがあれば、**`読み込み`** を押すと、あらかじめ保存した形状をこのユニットに取り込めます（一覧が空のときは、先に形状をプリセット登録する必要があります）。

### 3.2 形状の作成

1. キャンバスで **`形状編集`** を選び、**自由作図** では、画面案内 **「線作成: ドラッグして描画」**のとおり、**押したままドラッグ**して線分を引き、離すと1本の辺が確定します。  
2. 折れ線・多角形にするには、**辺を足すたびに同じくドラッグ**を繰り返します（前の頂点から続けて引くこともできます）。作った線の選択は案内 **「線選択: 線をクリック」**のとおりです。  
3. 資料と**同じ形**になっているか確認します。  
4. **Shift** を**押しながらドラッグ**すると、**水平・垂直**に揃いやすくなります（L字・直角向け）。

### 3.3 鉄筋配置

1. ユニット詳細の **`鉄筋配置`** で、鉄筋の位置（点）を追加します。
2. **D10**、**D13** など、呼び径（種類）を指定します。
3. 必要な**本数**分だけ配置します。

### 3.4 寸法（数値）と間隔線

キャンバス上のモードで **`寸法・間隔`** を選びます。画面上部の案内は次のとおりです。  
**ドラッグ: 間隔線を作成 / クリック: 寸法を追加 / 数値ラベル: ドラッグで移動**

- **クリック**で置いた箇所に、**寸法（mm）**を入力して **寸法**（注記）を足せます。形状長さの合算に使われます。
- **ドラッグ**で引いたのが **間隔線** です。ピッチ方向の目安として使い、線上の数値は主に本数まわりの解釈に使われます（画面の **寸法・間隔（要約）** でも確認できます）。

図を見ながら、区間ごとに資料の数値を入れてください。  
図面に**使う寸法だけ**入れ、参考寸法は **注記** の **`形状長さから除外する`** にチェックを入れて合算から外します（断面寸法など、経路長に含めない場合用）。

**入力・計算の目安**

- 足し込む例: `515`, `130`, `220`, `400` のように、形状パスに含める数値
- 除く例: 合算に入れない **参考・補助** 寸法
- L 曲がりがある場合は **`L字本数`** を入力します。1 以上で、合計から **20mm を 1 回**減じた扱いになります。
- 数値の取り違えは **`フック付`** 左側の**形状長さ**に影響するため、**形 → 寸法**の順で揃えると安全です。

**集計例**

```text
外周: 515 + 130 + 220 + 400 - 20 = 1245
内:   345 + 200 - 20 = 525
```

### 3.6 ピッチ

1. 詳細ツールバーで **`ピッチ`** モードに切り替え、**`@200`** の形式で **ピッチ**（mm）を入力します。  
   ピッチは **`フック付`** の**右側の本数**の計算に使われます。  
   また、**「ピッチは @200 のように指定します。」**
2. ピッチを空のまま **`保存`** しようとすると、**`ピッチを入力してください。`** という旨の保存前チェックで止まります。

### 3.7 保存

1. 入力が完了したら **`保存`** を押します。
2. **ユニット**一覧に戻り、行が表示されれば登録完了です。

### 3.8 長さプリセット

図面上で線を引くとき、よく使う **長さの候補** を呼び出すための **グループ** です。プロジェクトの図面編集で使います。**プロジェクトを作る前**に **`ユニット管理`** から登録しておくとスムーズです。

1. **`ユニット管理`** の右上、**`+ 新規作成`** の横の **`長さプリセット`** を押します。
2. **登録済みの長さグループを管理**する画面が開きます。未登録のときは **`追加`** から **`長さプリセットを追加`** ダイアログに進みます（空の場合は案内文どおり **「追加」** から作成）。
3. **`プリセット名`**（必須）と、任意で **`説明（任意）`**。続けて **`長さ一覧`** で **`番号`** と **`長さ（mm）`** を行ごとに入力。足りなければ **`+ 行を追加`** します（**`長さ（mm）`** は 1 行以上必須）。
4. **`保存`** で登録します。一覧の **`更新`** ・ **`削除`** であとから変更できます。
5. 図面では **`アクティブユニット`** 横のボタン（`title` は **`長さプリセットを選択`**。未選択時は表示 **`プリセットなし`**) で、線を引くたび使う候補グループを指定します。線を引くと **`長さを選択`** モーダルにプリセットの候補や **`任意の長さ`** 入力が出ます。画面案内どおり、クリックまたは**数字キー**で候補を素早く選べます。

## 4. プロジェクトの作成

ユニットのあと、図面と計算結果の単位となる **プロジェクト** を作ります。

1. ダッシュボード（**プロジェクト** 一覧）を開きます。
2. **`+ 新規プロジェクト`** を押します。
3. **`新規プロジェクト`** ダイアログで **`プロジェクト名`**（必須）と、必要なら **`説明`** を入れ、**`作成`** を押します（**`キャンセル`** で閉じられます）。
4. できた行を開き、プロジェクト詳細へ入ります。

## 5. 図面のアップロード

1. プロジェクト詳細のエリアに、図面アップロード枠（**`図面ファイルをドラッグ＆ドロップ、またはクリックしてアップロード`**）があります。
2. **`ファイルを選択`** から PDF / PNG / JPG（**最大 20MB**、アラート文のとおり）をアップロードします。
3. 完了すると、該当 **図面** の閲覧・編集URLへ進みます（1プロジェクト1図面登録。既に図面があるとアラート **「このプロジェクトにはすでに図面が登録されています。」** が出ます）。

## 6. 図面上で鉄筋線分を入れる

アップロード後、登録した **ユニット** と **長さプリセット** を使って線分を入れます。

1. 図面（ドローイング）画面を表示します。
2. **`アクティブユニット`** から、使う **ユニット**（色の代表）を選びます（プルダウン先頭に **`ユニットを選択してください`**）。
3. 同じ行のボタン（`title` **`長さプリセットを選択`**, 表示は **`プリセットなし`** または登録名）を押し、**`長さプリセットを選択`** モーダルで候補グループを選びます。使わない場合は、都度 `長さを選択`／直接入力に任せる運用も可能です（保存済みが無い場合の案内に従います）。
4. 図面上で、鉄筋区間を **クリック** するか **ドラッグ** して線分を描きます。**Shift** を押しながらなら、水平・垂直に制限しやすくなります。
5. **`長さを選択`** モーダルで **長さ (mm)**  を確定するか、**`任意の長さ`** に入力して確定します。**数字キー (1–9 など)** で、モーダル上の候補を速く選べる場合があります（画面案内 **「…長さをクリック、または数字キーで選択できます。」** ）。
6. 必要な区間を同様に繰り返します。

線分は最適化計算に反映されます。長さ・色・**アクティブユニット**の対応を確認してから、右側のパネルなどから **`切断計算`**（リンクで **`/projects/.../optimize` の計算画面**）へ進み、**`計算を実行`** までつなげてください（未入力のときは図面側でアラートが出る場合があります）。

## 7. 入力データの確認

計算画面（最適化ページ）の **`入力データ`** で、図面から取り込まれた要約を確認します。

- `外周` ／ `内` など、区分が合っているか
- 長さ (mm)  が合っているか
- 長さ（マーク）ごとの本数が合っているか
- 色（ユニット系統）が合っているか
- 合計本数が合っているか

違いがあれば、図面に戻り線分を直してから、再度 **切断計算** へ進みます。

## 8. 顧客情報

計算画面の **`顧客情報`** に、**`会社名`** ・**`住所`** ・**`顧客名`** を入れます（帳票の上段に出ます）。  
印刷時は、**`入力データ`** より上に並ぶ想定です。

## 9. 計算の実行

1. **`計算設定`** を確認します。
2. **`元材長さ (mm)`** に母材（原材）長を入れます（多くの場合 `6000`）。
3. **`部材長さ補正 (mm)`** を入れます（**マイナスで短く、プラスで長く**、例: `-30`。ラベル横の案内文のとおり）。
4. **`計算を実行`** を押すと、**`切断最適化を計算しています...`** のあと、**`計算結果`** が下に出ます。線分ゼロの場合は先に図面へ戻る必要があります（例: *計算対象の線分データがありません*）。

## 10. 計算結果

### 10.1 鉄筋種類別の必要本数

**`鉄筋種類別の必要本数`** を見て、**`長さ (mm)`** ごとの必要本数、**`D10`** など各呼びの列、**`合計`** を確認します（画面の表見出しに従います）。

### 10.2 フック付

**`フック付`** では、ユニット名ごとに、左が **形状の代表長さ (mm) 、右が本数** の積算表示となります（`1,245 × 202` などの形式）。

- **左**… ユニットの形状に対応する長さ（`フック付` 左列）
- **右**… 図面入力とピッチ等に基づく本数

本数は、内部で長さを 100mm 単位に丸めたあと、ピッチで求める扱いになっています。差があるときは **`入力データ`** やユニットの **ピッチ** から戻ります。

## 11. 印刷

1. 計算結果周辺の **`印刷`**（最適化結果UI）を押します。
2. ブラウザの印刷プレビューでレイアウトを確認し、**印刷** するか **PDF 保存** します。

### 日付やURLが付く場合

**ヘッダー／フッター** をオフにすると、日付・URL・ページ番号が出ないことが多いです（ブラウザの印刷ダイアログの設定に依存します）。

## 12. 作業手順（一覧）

1. ログインする
2. **`ユニット管理`** で **ユニット** を作り、色・形状・**寸法・間隔**・**ピッチ**・鉄筋を **`保存`** する
3. 使うなら **長さプリセット** も登録する（`長さプリセット` → `追加` → `保存` …）
4. プロジェクトを **作成** する
5. 図面を **アップロード** し、図面編集へ行く
6. **`アクティブユニット`** ・**長さプリセット**を選び、線分を入れる
7. **`切断計算`** から **入力データ** ・**顧客情報** ・**計算設定** を見て **`計算を実行`**
8. **`鉄筋種類別の必要本数`** ・**`フック付`** を確認し、**`印刷`** する

## 13. 想定外のときのチェック

### フック付の本数が違う

- 図面の数値（長さ）と本数
- 線の**色**＝**アクティブユニット**の系統
- ユニットの **ピッチ**（`@…`）と、ピッチ未入力の保存制限

### フック付左の形状長さが違う

- ユニットの **寸法** 注記と、**`形状長さから除外する`** の有無
- L曲がりと **`L字本数`**

### 図面で欲しいユニットが出ない

- ユニットを **`保存`** 済みか
- 無効化（一覧の無効扱い）にしていないか
- 色の重複で登録できない枠に当たっていないか

### 印刷の体裁

- 計算画面から **`印刷`** したか
- プレビューの**倍率**が適切か
- ヘッダー・フッターのオンオフ
